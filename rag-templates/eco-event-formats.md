# Event Announcement Formats
<!--
  Used by Ecosystem Agent: event_announce()
  One template per event type. Each generates short (≤280 char) and long (300-500w) variants.
  Delete these comment blocks before deploying.
-->

## PARTICIPATION (We're attending)

### Short form (Twitter/TG, ≤280 chars)
```
We'll be at [EVENT_NAME] in [CITY], [DATES]. 

🤝 Meet the team: [CALENDAR_LINK_OR_EMAIL]
📍 Find us: [BOOTH/LOCATION if applicable]

#[EVENT_HASHTAG] #[YOUR_PROJECT_TAG]
```

### Long form (Blog intro + details)
```
## [YOUR_PROJECT] at [EVENT_NAME] [YEAR]

[YOUR_PROJECT] is heading to [CITY] for [EVENT_NAME], [DATES].

[One paragraph: Why this event matters to us / what we're excited about]

**What we're doing there:**
- [Speaking slot / panel / workshop — with details]
- [Meeting with builders/investors/partners]
- [Demo / showcase if applicable]

**Want to connect?** [Book a meeting: LINK] or find us [WHERE].

We'll be sharing updates live from the event on [Twitter link]. Follow along.
```

## HOSTING (We're running it)

### Short form
```
We're hosting [EVENT_NAME] in [CITY], [DATE]. 

[ONE_LINE_WHAT_IT_IS — e.g. "A DePIN builder workshop for 50 developers."]

Register: [LINK]

#[YOUR_PROJECT_TAG]
```

### Long form
```
## [YOUR_PROJECT] is Hosting [EVENT_NAME]

[Date], [City / Online]

[One paragraph: Why we're hosting this, what problem it solves, who it's for]

**What to expect:**
- [Session 1 / Speaker / Topic]
- [Session 2]
- [Networking / Demo / Hackathon element]

**Who should attend**: [Target audience description]

**Registration**: [LINK] — [free/paid, capacity note if relevant]

Questions? [CONTACT]
```

## SPONSORSHIP

### Short form
```
Proud to sponsor [EVENT_NAME] [YEAR]. 

[ONE_LINE_WHY — e.g. "Supporting the builders shaping decentralized compute."]

See you in [CITY]. [LINK]

#[EVENT_HASHTAG] #[YOUR_PROJECT_TAG]
```

### Long form
```
## [YOUR_PROJECT] Sponsors [EVENT_NAME]

We're proud to announce our sponsorship of [EVENT_NAME], taking place [DATES] in [CITY].

[One paragraph: Why this event aligns with the project's mission]

[YOUR_PROJECT] will be [present with a booth / hosting a side event / sponsoring a specific track] at the event.

[Any specific activities — workshops, talks, demos]

**Visit us**: [Booth number / location details]
**Connect with our team**: [CALENDAR_LINK or contact method]
```

## HACKATHON

### Short form
```
[YOUR_PROJECT] is live at [HACKATHON_NAME] with [BOUNTY_AMOUNT] in prizes.

Build on [SPECIFIC_TRACK_OR_THEME]. 

Details + register: [LINK]

#[HACKATHON_HASHTAG] #[YOUR_PROJECT_TAG] #buidl
```

### Long form
```
## Build on [YOUR_PROJECT] at [HACKATHON_NAME]

[HACKATHON_NAME] is here — and [YOUR_PROJECT] is offering [BOUNTY_AMOUNT] for the best builds on our [API/SDK/PROTOCOL].

**Bounty tracks:**
- [Prize 1]: [AMOUNT] — [Description of what qualifies]
- [Prize 2]: [AMOUNT] — [Description]
- [Prize 3 / Honorable mentions]: [AMOUNT]

**Resources for builders:**
- Docs: [LINK]
- SDK: [LINK]
- Example projects: [LINK]
- Dev support: [Discord channel link]

**Judging criteria**: [Criteria 1], [Criteria 2], [Criteria 3]

Submissions close: [DATE]

Questions? Join [#hackathon Discord channel] or DM [@HANDLE].
```
